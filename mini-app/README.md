# Mini App

Telegram Mini App на React + TypeScript + Telegram WebApp SDK.

## Layout

```
mini-app/
├── src/
│   ├── components/
│   ├── pages/
│   ├── hooks/
│   ├── services/      # API клиент
│   ├── store/         # state
│   └── main.tsx
├── public/
└── package.json
```

## Quickstart

> ⚠️ Заготовка. Реализация будет добавлена через issue по Phase 2.

```bash
npm install
npm run dev
```

Сборка: `npm run build`.
