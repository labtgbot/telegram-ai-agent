# Press kit — assets checklist

Drop the files below into this folder before the public launch.
Everything is **optional** for the technical launch (issue #37); we
keep the checklist in version control so the marketing pass has a
single source of truth.

## Required for Product Hunt / IndieHackers

- [ ] `logo.svg` — vector logo, 1:1 aspect ratio.
- [ ] `logo-mark-1024.png` — 1024×1024 transparent PNG.
- [ ] `cover-1280x720.png` — Product Hunt cover image.
- [ ] `screenshot-bot-chat.png` — bot replying to `/ask`.
- [ ] `screenshot-mini-app.png` — Mini App balance + buy screen.
- [ ] `screenshot-admin-crm.png` — CRM dashboard hero shot.
- [ ] `founder-bio.md` — 80–120 word bio used in interviews.
- [ ] `contact.txt` — public press contact email.

## Optional

- [ ] `demo-video.mp4` — 60-second narrated demo.
- [ ] `pitch-deck.pdf` — investor / partner deck.
- [ ] `og-card-1200x630.png` — Open Graph image for the marketing
      page (if a marketing site exists).

## Format constraints

- Maximum file size 5 MB each (Product Hunt cap).
- Screenshots must show **real** production data with PII redacted
  via the same scrubber rules as `docs/security/pii-minimization.md`.
- Logos use the brand palette in `docs/architecture/diagrams/` (the
  same blue used in the architecture diagrams).
